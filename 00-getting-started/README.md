# Getting Started

This section introduces the **Green Deal Data Space (GDDS) Rulebook** and provides the context needed to navigate the rest of the document. New readers should start here before moving on to the building blocks.

It covers:

- the **Preamble**, setting out the scope and intent of the Rulebook,
- an **Introduction to the GDDS**, explaining its purpose and relationship to the European Green Deal,
- the **Rulebook itself**, describing how it is structured and how it should be applied,
- the **Documentation structure**, describing how the chapters are organised,
- and the **Glossary**, providing definitions for the terminology used throughout.
